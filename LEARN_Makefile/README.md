## README

**Contents of `Makefile`:**

```
start:
        echo Program starts!

test:
        echo Testing starts now!

help:
        echo All you need is to believe in yourself.
```

**now you can run on terminal:**
```bash
make start
# Output: 
# echo Program starts!
# Program starts!

make test
# Output: 
# echo Testing starts now!
# Testing starts now!

make help
# Output: 
# echo All you need is to believe in yourself.
# All you need is to believe in yourself.
```
